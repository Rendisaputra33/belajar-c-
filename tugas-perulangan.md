### **TUGAS PEMROGRAMAN DASAR C++**

**Materi: Perulangan `while`**

---

#### **Studi Kasus: Kasir Sederhana**

Buatlah sebuah program kasir sederhana menggunakan bahasa **C++** dengan ketentuan sebagai berikut:

1. Program meminta pengguna memasukkan **harga barang**
2. Program menjumlahkan setiap harga barang yang dimasukkan
3. Setelah memasukkan harga barang, program menanyakan:

   ```
   Tambah barang lagi? (y/n):
   ```
4. Jika pengguna memasukkan `y`, program mengulang input harga barang
5. Jika pengguna memasukkan `n`, program berhenti
6. Setelah program berhenti, tampilkan **total belanja**

---

#### **Contoh Interaksi Program**

```
Masukkan harga barang: 5000
Tambah barang lagi? (y/n): y
Masukkan harga barang: 3000
Tambah barang lagi? (y/n): y
Masukkan harga barang: 2000
Tambah barang lagi? (y/n): n

Total belanja: 10000
```

---

#### **Ketentuan Teknis**

* Gunakan **perulangan `while`**
* Gunakan tipe data yang sesuai
* Gunakan variabel untuk menyimpan total belanja
* Gunakan `char` untuk pilihan (`y` atau `n`)
* Program ditulis menggunakan bahasa **C++**
